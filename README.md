# MY TRAVEL TYPING ｜ 나의 유럽 여행 앨범 ✈️

[tw-metro-typing](https://github.com/ridemountainpig/tw-metro-typing)의 타이핑 엔진 구조를 바탕으로,
디자인부터 게임성, 타이핑 엔진까지 **감성적이고 완벽한 여행 타이핑 게임**으로 대대적으로 업그레이드한 프로젝트입니다.

단순한 디지털 인터페이스를 벗어나, **부드러운 크림색 배경과 앨범 사진, 여권 스탬프와 항공권**을 모티브로 한 따뜻한 감성(아날로그 앨범 테마)을 제공합니다.

## ✨ 주요 기능 및 업그레이드 내역

### 1. 아름다운 디자인 & UI/UX ("나의 유럽 여행 앨범" 컨셉)
- **감성 디자인 시스템:** 클래식한 세리프 폰트와 모던 산세리프의 조화, 따뜻한 색상 팔레트 적용.
- **홈 화면 개편:** 가로 스크롤 카드 형태의 코스 갤러리와 하단 설정 시트(Bottom Sheet)로 게임 진입 시의 매끄러운 UX.
- **도착 팝업 (Arrival Popup):** 목적지 도착 시 화면을 덮는 앨범 사진과 종이 테이프 연출로 극적인 달성감 제공.
- **항공권 스코어바:** 게임 플레이 화면 상단을 여권/항공권 스탬프 스타일의 UI로 개편.

### 2. 게임성 및 피드백 강화
- **콤보 시스템:** 오타 없이 연속으로 입력하면 콤보(🔥)가 쌓이며, 오타 시 콤보가 초기화됩니다. 최종 결과에 '최고 콤보'가 기록됩니다.
- **타이핑 손맛 (Particle Pop):** 정확하게 입력된 글자는 초록빛(Teal)으로 통통 튀어오르는 마이크로 애니메이션 적용.
- **오타 시각 피드백:** 오타 시 화면이 흔들리며 타이핑 대상 글자가 일시적으로 빨간색(Coral)으로 변합니다.
- **세분화된 난이도 시스템:**
  - 🌱 **초급 (한글 지명):** 가볍고 편안한 타이핑
  - 🧭 **중급 (한글 + 영문 힌트):** 지명 학습
  - 🔥 **상급 (영문 지명 + 블라인드 모드):** 목표 단어의 첫 글자만 힌트로 노출되고 나머지는 숨겨져(`•`), 철자를 외우거나 직접 유추해서 타이핑하는 하드코어 모드.

### 3. 타이핑 엔진 & 기술적 고도화
- **유연한 오타 매칭:** 외래어 표기법 상 헷갈리기 쉬운 모음/자음을 유연하게 허용 (`FLEXIBLE_MATCHES`). 예: '쾨'→'코', '스'→'쑤', '르'→'러' 등.
- **상태 관리 최적화:** Zustand 전역 스토어(`useGameStore`)를 도입하여 복잡한 설정과 게임 진행 상태를 깔끔하게 분리.
- **지도 애니메이션 개선:** 버스는 완벽한 직선(0), 비행기는 역동적인 곡선 항로와 회전 애니메이션 적용.

## 🛠️ 기술 스택

- Vite 5 + React 18
- 상태 관리: **Zustand**
- 스타일: 순수 CSS (CSS Variables, Flex/Grid 레이아웃)
- 아이콘: **lucide-react**
- 지도 시각화: **d3-geo** + **topojson-client**

## 🚀 로컬 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://127.0.0.1:5173` 접속.

프로덕션 빌드:

```bash
npm run build
npm run preview
```

## 🗺️ 코스 추가 및 데이터 구조
`public/data/travel-routes.json` 파일에 코스와 정거장을 추가하면 지도와 게임에 자동으로 반영됩니다.

- 국가(country)가 동일하면 **버스(직선 이동)**로, 국가가 변경되면 **비행기(곡선 항로)**로 이동수단이 **자동 판정**됩니다 (`src/lib/geo.js`).
